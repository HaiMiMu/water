# 🦫 卡皮巴拉喝水小助手

一个温柔提醒你喝水的桌面小工具，以可爱的卡皮巴拉为主题。

## 下载

👉 [立即下载](capybara-water-reminder.exe) (约 43 MB, Windows 10/11 64位)

## 功能特点

- ⏰ **定时提醒** — 可自定义 15/30/45/60 分钟提醒间隔
- 💧 **快速记录** — 100/250/500/1000ml 一键记录 + 自定义水量
- 📊 **进度追踪** — 实时显示今日喝水进度，支持自定义目标
- 🔥 **连续打卡** — 记录连续达成目标天数
- 📅 **日历热力图** — 按月查看喝水记录，柱状图显示时间分布
- 🌙 **深色主题** — 一键切换浅色/深色模式
- 🖥️ **系统托盘** — 最小化到托盘，右键快速操作
- 🔄 **开机自启** — 支持随 Windows 启动
